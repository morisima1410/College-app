📱 College Management App
📖 Description:

This is a comprehensive College Management Android Application designed to digitize academic and administrative processes. It provides students with a centralized platform to access important information such as notifications, results, study materials, and more.

🔥 Features:

User Notifications
Question Papers
Results System
Faculty List
Placement Form
Gallery
Login & Registration
User Profile
Lab Information
Admission Form
Feedback System
Location Sharing
Developer Information
💻 Technologies:
Android Studio
Java 
Firebase
